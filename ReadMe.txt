School Fee Management Dashboard (Excel Project)

This project is a simple school fee management dashboard built using Microsoft Excel. It helps track student payments, calculate balances, and provide clear insights through charts and summaries.
The aim of this project is to:
I.	Track student fee payments
II.	Identify unpaid or partially paid students
III.	Present data in a simple and visually appealing dashboard

📊 Features
I.	Automatic balance calculation
II.	Automatic fee status
III.	Pivot tables for summary
IV.	Charts for visualization
V.	Interactive filters (slicers)
VI.	Clean dashboard layout

🧮 How It Works
1. Enter student data in the worksheet as defined by the column:
I.	Name
II.	Class
III.	Amount Paid
IV.	Total Fee

2. The system automatically:
I.	Calculates balance
II.	Assigns fee status

▶️ How to Use the File
1. Open the Excel file
2. Go to the Student Data sheet
3. Add or update student payment details
4. Click Refresh All (Data tab)
5. Go to the Dashboard sheet to view results

▶Tools Used
I.	Microsoft Excel
II.	Pivot Tables
III.	Excel Formulas
IV.	Charts and Slicers


▶Use Case
This dashboard can be used by:
I.	Schools
II.	Training centers
III.	It can also be modified for other financial tracking purposes.
